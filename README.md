## Teslo API

1. Clonar proyecto

2. `npm i`

3. Clonar el archivo `.env.template` y renombrarlo a `.env`

4. Cambiar las variables de entorno

5. Levantar la base de datos

```
docker-compose up -d
```

6. Levantar: `npm run start:dev`

7. Ejecutar Seed

```
http://localhost:3000/api/seed
```
